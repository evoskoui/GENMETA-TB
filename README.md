# GENMETA-TB
## Genomics and Metabolomics of *Mycobacterium tuberculosis* - An Integrated Analysis

This directory contains code for the data analysis within, "Metabolomics profiling reveals changes in peptidoglycan and redox metabolism between ancient and modern Mycobacterium tuberculosis."

All data files contained in this repository are examples, and the main data files used are provided with the paper.
All scripts are provided to detail what is described in the methods section "Metabolomics Data Analysis".
This repository was created to store the methodology in an organized and accessible fashion.
